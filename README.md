<h2>Usage</h2>

<h3>Installation</h3>
<pre lang="bash">
$ git clone https://github.com/endygamedev/network-manager-gui
$ cd network-manager-gui
$ sudo make install
</pre>

<h3>Update</h3>
<pre lang="bash">
$ sudo make update
</pre>

<h3>Uninstall</h3>
<pre lang="bash">
$ sudo make uninstall
</pre>

<h3>Run</h3>
<pre lang="bash">
$ network-manager-gui
</pre>
